# KwynthVPN

Готовые установочные файлы KwynthVPN для Windows и Android.

## Скачать

- **Windows:** `KwynthVPN-Setup.exe`
- **Android:** `KwynthVPN.apk`

## Быстрый старт

### Windows
1. Скачайте `KwynthVPN-Setup.exe`
2. Запустите установщик
3. Откройте KwynthVPN и импортируйте свою VLESS-ссылку

### Android
1. Скачайте `KwynthVPN.apk`
2. При необходимости разрешите установку из неизвестных источников
3. Установите приложение и импортируйте свою VLESS-ссылку

## Если не подключается

- Проверьте, что VLESS-ссылка скопирована полностью
- Для Windows запускайте приложение от имени администратора (особенно для туннельного режима)
- Попробуйте переключить режим VPN и подключиться снова
